

# Belief Propagation Simulator

## Description
The **Belief Propagation Simulator** is a tool designed to simulate the belief propagation algorithm, which is widely used in graphical models such as Bayesian networks and Markov random fields.

## Installation
To install the necessary dependencies, run the following command:

```bash
pip install -r requirements.txt
```

## Usage
Here is an example of how to run the simulator:

```bash
python simulator.py --input example_input.json --output result.json
```

### Parameters
- `--input`: Path to the input file containing the graphical model.
- `--output`: Path to the file where the simulation results will be saved.

## Example
```bash
python simulator.py --input sample_graph.json --output output_results.json
```
